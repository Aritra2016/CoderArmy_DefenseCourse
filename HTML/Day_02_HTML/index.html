Collections Framework Concept

1. What is the Java Collections Framework?

  • The Java Collections Framework provides a set of interfaces and classes to store and manipulate groups of objects. It includes interfaces like List, Set, Map, and implementations like ArrayList, HashSet, and HashMap.

    • 2. What is the difference between Collection and Collections in Java?

    1. Collection is an interface that represents a group of objects.
    2. Collections is a utility class that provides static methods to operate on collections, such as sorting or searching.

    Explain the difference between List, Set, and Map.

    1. List allows duplicate elements and maintains insertion order.
    2. Set does not allow duplicates and may or may not maintain order (e.g., HashSet doesn't, LinkedHashSet does).
